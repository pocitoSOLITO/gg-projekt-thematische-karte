## Arbeitsauftrag: Erstellung einer thematischen Karte

Erstellen Sie eine thematische Karte. Dies kann analog oder digital geschehen, je nach Vorliebe.  
Leere Vorlagen können Sie in A4, A3 oder A2 (letzteres zum Zusammensetzen) beziehen oder per E-Mail anfragen und digital erhalten.

---

### Schritte

- **Suchen Sie nach einem interessanten und geeigneten Thema.**
- **Überlegen Sie sich**, welche Indikatoren (mindestens zwei, um einen möglichen Zusammenhang zu erkennen) Sie darstellen möchten  
  *(grafisch, nicht als Zahlenwerte).*  
- **Finden Sie die benötigten Daten und Fakten.**
- **Überlegen Sie sich, wie Sie die Karte gestalten möchten:**
  - welcher geographische Raum soll untersucht werden?
  - wie sollen die Indikatoren dargestellt werden?
  - sollen absolute oder relative Werte verwendet werden?
  - welche Kategorien sind für die Indikatoren sinnvoll?
- **Erstellen Sie die Karte.**
- **Analysieren Sie die Aussage Ihrer Karte** und formulieren Sie diese schriftlich (ca. ½ Seite).  
  Es ist möglich, dass Ihre beiden Indikatoren keinen Zusammenhang erkennen lassen — auch das kann eine sinnvolle Aussage sein!
- **Verfassen Sie eine kurze Reflexion** über Verlauf, Resultat und Lernerfolg (ca. ½ Seite) — mit etwas Distanz zu Ihrem Arbeitsprozess.
